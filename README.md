# X-Clone 🚀

A fully responsive clone of the X (Twitter) built with **HTML** and **Tailwind CSS**. This project focuses on modern UI/UX, mobile-first design.

👉 **[Live preview](https://codamee.github.io/x-clone-tailwind)**

## ✨ Features

* **Mobile Design:** Fully responsive layout that adapts from 300px phones to large desktops.
* **Tailwind CSS:** Utilizes utility-first styling for high performance and custom HSL color palettes.
* **Material Symbols:** Integrated Google Material Symbols for a modern icon set.
---

## 🛠️ Tech Stack

* **Frontend:** HTML5, Tailwind CSS
* **Icons:** Google Material Symbols (Variable Fonts)
